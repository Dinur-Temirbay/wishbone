# Wishbone — Landing Page 🦴

Современный адаптивный лендинг, выполненный на чистом стеке с использованием препроцессора SCSS. Проект демонстрирует навыки компонентного подхода к верстке и создания отзывчивых интерфейсов.

## 🛠 Технологический стек

- **HTML5:** Семантическая разметка.
- **SCSS:** Модульная архитектура (БЭМ-подобная структура в `base/` и `blocks/`), использование переменных и миксинов.
- **CSS3:** - Построение сеток на **Flexbox/Grid**.
    - **Media Queries** для реализации адаптивной верстки (Mobile/Tablet/Desktop).
- **JavaScript:** Базовая логика взаимодействия (Vanilla JS).

---

## 🚀 Как запустить проект

Для корректной работы всех функций (особенно внешних скриптов и путей) рекомендуется использовать локальный сервер.

Использование Live Server (рекомендуется):

1. Откройте папку проекта в **VS Code**.
2. Запустите через расширение **Live Server**.
3. Проект откроется по адресу `http://127.0.0.1:5500`.

---

## 📱 Особенности реализации

- **Fully Responsive:** Полная адаптация интерфейса под мобильные устройства, планшеты и широкие экраны.
- **Modular Architecture:** Стили разбиты на независимые компоненты, что обеспечивает чистоту и масштабируемость кода.
- **Zero Dependencies:** Проект реализован на нативных технологиях без использования тяжелых CSS-фреймворков (Bootstrap, Tailwind и др.).
